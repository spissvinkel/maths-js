External module "vec3"

vec3 is a collection of functions to manipulate Vec3 3-element vector objects.

The primary use for a Vec3 object is in 3D geometry, as homogeneous coordinates in 2D geometry, or to represent e.g. RGB colours.

Const lerp2

  • lerp2(a1: Vec3, b1: Vec3, a2: Vec3, b2: Vec3, s: number, t: number, c: Vec3): Vec3
    • Defined in vec3.ts:509

    Bilinear interpolation between a1, b1, a2 and b2 based on s and t, where s and t are numbers between 0.0 and 1.0. The result is stored in c

    The calculation is as follows: a1 and b1 are interpolated based on s to give p, a2 and b2 are interpolated based on s to give q, and then the final result is obtained by interpolating p and q based on t.

    The result will be equal to a1 when both s and t is 0.0, equal to a2 when s is 0.0 and t is 1.0, equal to b1 when s is 1.0 and t is 0.0, and equal to b2 when both s and t is 1.0

    Parameters

    • a1: Vec3

      the first start value - a 3-element vector object

    • b1: Vec3

      the first end value - a 3-element vector object

    • a2: Vec3

      the second start value - a 3-element vector object

    • b2: Vec3

      the second end value - a 3-element vector object

    • s: number

      a floating point number in the interval [0.0, 1.0]

    • t: number

      a floating point number in the interval [0.0, 1.0]

    • c: Vec3

      a 3-element vector in which to store the result

    Returns Vec3

    c - the interpolation result
